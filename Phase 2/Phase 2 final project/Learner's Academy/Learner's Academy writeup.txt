Project Writeup

Project Name:Learner's Academy

Project Description:
Created a application for school that has an online management system. The system keeps track of its classes, subjects, students, and teachers. 
It has a back-office application with a single administrator login.

 The administration can do the following things via this application:
●  list of all the subjects for all the classes
●  list of all the teachers
●  list of all the classes
● Assign classes for subjects from the list
● Assign teachers to a class for a subject (A teacher can be assigned to different classes for different subjects)
● Get a master list of students (Each student must be assigned to a single class)
